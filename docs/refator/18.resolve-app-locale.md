# 18.resolve-app-locale

## 发现的问题

route query locale 解析在 navigate / LanguageSwitcher / LocalizedLink 三处重复。

## 问题证据和影响范围

| 项 | 说明 |
| ---- | ------ |
| 证据 | 三处相同三元；与 `normalizeLocale` 模糊映射不同 |
| 影响 | `settings.ts` + 三组件 |
| 行为 | 不变（严格 exact match） |

## 最小修改方案

```ts
export function resolveAppLocale(queryLocale: unknown): AppLocale {
  return isSupportedLocale(queryLocale) ? queryLocale : DEFAULT_LOCALE;
}
```

## Design Challenger 结论

- **verdict:** APPROVE（必须改三处）

## 最终修改

- 新函数 + 三处替换 + tests
