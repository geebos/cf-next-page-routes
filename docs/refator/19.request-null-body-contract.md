# 19.request-null-body-contract

## 发现的问题

Tauri `request()` 对 204/304 必须传 `null` body，否则 `Response` 抛错；无测。

## 问题证据和影响范围

| 项 | 说明 |
| ---- | ------ |
| 证据 | WHATWG；生产 `hasNoBodyStatus` |
| 影响 | 新增 `request.test.ts` |
| 生产 | 零改动 |

## 最小修改方案

- 非 Tauri → throw（真路径）
- 204/304 `Response` 构造契约（因 `mock.module` 不可用，未 mock invoke）

## Design Challenger 结论

- **verdict:** APPROVE（mock invoke）；落地因环境降级为契约测

## Code Reviewer 结论

- **APPROVED**（NON_BLOCKING：未执行 request 内 204 分支）

## 验证结果

`pnpm test` → **90 pass**。
