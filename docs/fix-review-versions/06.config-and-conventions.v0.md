## Implementation steps

### 1. Trailing slash policy (required)

Next config:

```ts
trailingSlash: true,
```

`localizePath` already emits trailing `/`.

Wrangler assets: change away from drop if the product wants trailing slash, e.g.:

```jsonc
"html_handling": "force-trailing-slash"
// or another mode that does not 308 away from /en/todo/
```

Verify against current Cloudflare docs for the wrangler major in `package.json`. After change, request `/en/todo/` and `/en/todo` and confirm intended redirect direction (or 200).

### 2. Todo page Section (required)

In `src/pages/[locale]/todo.tsx`:

- Import `Section` from `@/components/ui/section` (same as test page).
- Replace create-form and list wrappers:

```tsx
<Section className="rounded-lg border border-border bg-card p-4">
  <CreateTodoForm ... />
</Section>

<Section>
  <List ... />
</Section>
```

Preserve existing spacing/classes as needed for visual parity.
